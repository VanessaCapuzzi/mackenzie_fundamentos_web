Sobre Banco de Dados

Bancos de dados, como são denominados em português, também conhecidos como bases de dados, são conjuntos de dados relacionados entre si com registros dos mais diversos tipos. São muito importantes para empresas e há muito se tornaram de grande valia para os sistemas de informação e segurança.

Os Sistemas Gerenciadores são responsáveis por operar o Bancos de Dados, desde os anos 70. Antes disso, havia os sistemas de arquivos do sistema operacional, para armazenamento destas informações. Nos anos 80, a tecnologia de SGBD relacional passou a dominar o mercado, e hoje em dia é utilizada em praticamente todos os bancos de dados.

O sistema de Banco de Dados atua principalmente no controle de operações empresariais. Outra de suas importantes funções é gerenciar as informações de estudos.
